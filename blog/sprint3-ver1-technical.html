Html, Css and Javascript - An Analogy

Here's my analogy of HTML, CSS and Javascript.:

HTML is like a four-chord progression in a song. Let's say, we have a G - C - Em - D. This four chords set the structure. In programming, HTML lays the foundation of a web page.

CSS is like the pentatonic scale played over the four-chord song. It adds different layer of notes to the song and make it sound more beautiful. CSS makes a web page much more visually appealing to the viewer and user of the page.

JAVASCRIPT is like how the guitar player phrase the guitar notes together to form a repetitive tagline of a song. Javascript makes a web page interactive just like the guitar intro of the famous rock song, "Sweet Child Of Mine" by Guns & Roses.
