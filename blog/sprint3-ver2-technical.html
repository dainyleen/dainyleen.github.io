What is a Javascript Function?

Is a set of instructions that we want to run over and over again.

This is how a javascript function works:

  1. Declare a function. We can use the function keyword.
  2. Name a function. To name a function, we follow the rules for naming a variable.
  3. The name is followed by a parenthesis. This is how we can tell that we are dealing with a function.
  4. Include the curly braces which form the code block. The javascript we place inside the curly brace runs whenever the program activates the function.
  5. To activate the code inside a function, we call the function.

Functions provide a way to execute code that we want to repeat again and again on our web page.
