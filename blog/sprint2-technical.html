ID versus CLASS

CSS Selectors are used to target html elements on a web page that we want to style.

ID Selector

IDs use a hashtag (#) at the beginning of each declaration.

IDs are unique to a page. It is important to remember that a page can only have one element with the same ID name.

The example below shows an incorrect use of an ID selector.

CLASS Selector

Classes use a full-stop (.) at the beginning of each declaration.

Classes are more flexible than ID selectors. We can use classes when we know a selector will not be unique.
